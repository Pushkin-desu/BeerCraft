# BeerCraft

Brewery-lite плагин для Paper 1.21.11+. Вари кастомное пиво в ванильных котлах.

[README in English](README.md)

## Функционал

- 11 рецептов в ванильных котлах
- Определение источника тепла (костёр, душевой костёр, огонь, душевой огонь, лава, магма)
- Кастомные зелья через PotionMeta + CustomModelData (нужен ресурс-пак для текстур)
- Визуальный и звуковой фидбек (частицы, звуки, сигнал готовности и переварки)
- Состояние котла хранится в chunk PDC (переживает рестарт)
- Двуязычные сообщения (en/ru)
- Перезагрузка конфига и рецептов через `/beercraft reload`

## Установка

1. Скачай `beercraft-X.Y.Z.jar` со страницы Releases.
2. Положи в папку `plugins/`.
3. Перезапусти сервер.
4. Отредактируй `plugins/BeerCraft/config.yml` и `recipes.yml` по необходимости.
5. `/beercraft reload` для применения изменений.

## Команды

| Команда | Право | Описание |
|---------|------|----------|
| `/beercraft reload` | `beercraft.admin` | Перезагрузить конфиг и рецепты |
| `/beercraft list` | `beercraft.admin` | Список всех рецептов |
| `/beercraft give <рецепт> [игрок]` | `beercraft.admin` | Выдать бутылку пива |
| `/beercraft info` | `beercraft.admin` | Инфо о плагине |

Право `beercraft.admin` выдаётся операторам по умолчанию.

## Варка

1. Поставь котёл и залей водой с помощью ведра.
2. Поставь источник тепла снизу (горящий костёр, огонь, источник лавы и т.д.).
3. Правый клик по котлу каждым ингредиентом (по одному предмету за клик).
4. Жди. Зелёные частицы + звук повышения уровня — варка готова, разливай.
5. Правый клик пустой стеклянной бутылкой — забрать пиво.
6. Выпить бутылку — получить эффекты.

Если ингредиенты не соответствуют рецепту или время варки вышло за окно — будет бурда. Бурда без эффектов.

## Конфигурация

```yaml
cauldron-tick-period: 20          # Тиков между инкрементами (20 = 1с, требует рестарт)
boil-time-min-multiplier: 0.8     # Минимальный множитель времени варки
boil-time-max-multiplier: 1.5     # Максимальный множитель времени варки
failed-brew-action: drop_swill    # drop_swill | nothing
state-flush-period-seconds: 60    # Частота сохранения (требует рестарт)
heat-sources: [CAMPFIRE, ...]     # Источники тепла
locale: ru                        # en | ru
bstats: true
```

> **Примечание:** `cauldron-tick-period` и `state-flush-period-seconds` применяются только после перезапуска сервера. Остальные параметры применяются через `/beercraft reload`.

## Сборка

## Функционал

- Залей котёл водой и поставь источник тепла снизу
- Правый клик ингредиентом по котлу добавляет ингредиент
- При наличии источника тепла котёл варит
- Правый клик стеклянной бутылкой — собрать пиво
- Совпадение с рецептом → кастомное зелье с эффектами
- Несовпадение → «Бурда»
- Выпить пиво — получить эффекты
- Состояние котла сохраняется в chunk PDC, переживает рестарты

## Источники тепла

Настраиваются в `config.yml`. По умолчанию: Костёр (должен гореть), Душевой костёр (должен гореть), Огонь, Душевой огонь, Лава (только источник), Магма.

## Рецепты

Определяются в `recipes.yml`. Три встроенных рецепта:

| Название | Ингредиенты | Варка | Эффекты |
|----------|-------------|-------|---------|

| Тёмное нефильтрованное | 4 Пшеница, 2 Какао-бобы | 45с | Сила I (45с), Медлительность I (30с) |
| IPA | 3 Пшеница, 2 Ягоды, 1 Сахар | 60с | Проворство II (90с) |

## Конфигурация

```yaml
cauldron-tick-period: 20          # Тиков между инкрементами (20 = 1с)
boil-time-min-multiplier: 0.8     # Минимальный множитель времени варки
```

> **Примечание:** `cauldron-tick-period` и `state-flush-period-seconds` применяются только после перезапуска сервера. Остальные параметры применяются через `/beercraft reload`.
